### Software Requirements

<CheckList
  items={[
    "Windows 8.1, Windows 10, Windows 11, or Windows Server 2019+ (fully updated)",
    "Visual C++ Redistributable 2019 or newer",

  ]}
/>

<InfoBanner type="warning" title="Important">
  FiveM requires a **fully updated** version of Windows. An outdated operating
  system may not work correctly. Windows 10 or newer is recommended for the best
  experience.
</InfoBanner>
