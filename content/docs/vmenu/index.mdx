vMenu is a powerful, feature-rich menu system for FiveM servers that provides comprehensive server administration tools and customizable player options. Developed by Vespura, it's one of the most popular and reliable menu systems in the FiveM community.

## Overview

vMenu offers an extensive set of features for both server administrators and players, with granular permission control and extensive customization options.

### Key Features

**Player Management:**
- **Teleportation** - Waypoint, coordinates, saved locations
- **Vehicle Control** - Spawn, customize, and manage vehicles
- **Weapon Management** - Weapon loadouts and customization
- **Player Options** - God mode, invisibility, fast run/swim
- **Appearance Editor** - Complete character customization

**Server Administration:**
- **Player Moderation** - Kick, ban, warn, spectate players
- **Server Management** - Weather control, time synchronization
- **Vehicle Management** - Spawn and manage server vehicles
- **Permission System** - Granular control over features
- **Voice Chat Control** - Proximity and radio channel management

**Advanced Features:**
- **Recording Mode** - Hide UI elements for content creation
- **Saved Locations** - Personal and server-wide teleport points
- **Vehicle Persistence** - Save and respawn custom vehicles
- **Ban Management** - Comprehensive ban system with appeals
- **Multi-Language Support** - Translations for global communities
