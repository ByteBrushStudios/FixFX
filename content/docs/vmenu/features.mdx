## Overview

vMenu provides a comprehensive set of features for server administration, player management, and customization. This guide covers all available features organized by menu category.

<Callout type="info">
  Features shown depend on your permissions. Admins see all options, while regular players see limited features based on configuration.
</Callout>

## Main Menu Categories

| Category | Description | Permission Required |
|----------|-------------|---------------------|
| **Online Players** | Manage connected players | `vMenu.OnlinePlayers` |
| **Player Options** | Self modification options | `vMenu.PlayerOptions` |
| **Vehicle Spawner** | Spawn any vehicle | `vMenu.VehicleSpawner` |
| **Vehicle Options** | Modify current vehicle | `vMenu.VehicleOptions` |
| **Saved Vehicles** | Spawn saved vehicles | `vMenu.SavedVehicles` |
| **Personal Vehicle** | Personal vehicle management | `vMenu.PersonalVehicle` |
| **Weapon Options** | Weapon management | `vMenu.WeaponOptions` |
| **Weapon Loadouts** | Save/load weapon sets | `vMenu.WeaponLoadouts` |
| **Player Appearance** | Character customization | `vMenu.PlayerAppearance` |
| **Saved Characters** | Saved appearance presets | `vMenu.SavedCharacters` |
| **Teleport Locations** | Teleport to locations | `vMenu.Teleport` |
| **World Options** | Change time/weather | `vMenu.WorldOptions` |
| **Voice Chat** | Voice chat settings | `vMenu.VoiceChat` |
| **Misc Settings** | General settings | `vMenu.MiscSettings` |
| **Recording** | Rockstar Editor controls | `vMenu.Recording` |

## Online Players

**Permission**: `vMenu.OnlinePlayers`

Manage connected players on your server.

### Player List Features

- **View All Players**: See complete list of connected players
- **Player Count**: Display current player count
- **Server Info**: View server details

### Player Actions

**Permission**: `vMenu.OnlinePlayers.All` or specific sub-permissions

| Action | Permission | Description |
|--------|------------|-------------|
| **Teleport To** | `.TeleportToPlayer` | Teleport to player's location |
| **Summon** | `.Summon` | Bring player to you |
| **Kill** | `.Kill` | Kill the player |
| **Kick** | `.Kick` | Kick from server |
| **Temp Ban** | `.TempBan` | Temporary ban (hours/days) |
| **Perm Ban** | `.PermBan` | Permanent ban |
| **Spectate** | `vMenu.Spectate` | View player's screen |
| **Freeze** | `.Freeze` | Freeze player movement |
| **Unfreeze** | `.Freeze` | Unfreeze player |
| **Remove Weapons** | `.RemovePlayerWeapons` | Strip all weapons |
| **View Identifiers** | `.SeePlayerIdentifiers` | Show steam, license, discord IDs |

### Player Information Display

- Current health and armor
- Current vehicle (if in vehicle)
- Current weapon
- Distance from you
- Player identifiers

<Callout type="warn">
  **Moderation Warning**: Always follow your server's moderation guidelines when using player management features.
</Callout>

## Player Options

**Permission**: `vMenu.PlayerOptions`

Modify your own character's abilities and status.

### God Mode Options

| Feature | Permission | Description |
|---------|------------|-------------|
| **God Mode** | `.God` | Invincibility (no damage) |
| **Invisible** | `.Invisible` | Become invisible to players |
| **Never Wanted** | `.NeverWanted` | Disable wanted level |
| **Ignored By Police** | `.IgnoredByPolice` | Police won't chase you |
| **Set Wanted Level** | `.SetWanted` | Manual wanted level (1-5 stars) |

### Movement Options

| Feature | Permission | Description |
|---------|------------|-------------|
| **Fast Run** | `.FastRun` | 2x running speed |
| **Fast Swim** | `.FastSwim` | 2x swimming speed |
| **Super Jump** | `.SuperJump` | Jump extra high |
| **Unlimited Stamina** | `.UnlimitedStamina` | Never get tired |
| **No Ragdoll** | `.NoRagdoll` | No falling animations |

### Health & Armor

| Feature | Permission | Description |
|---------|------------|-------------|
| **Restore Health** | `.MaxHealth` | Heal to full |
| **Restore Armor** | `.MaxArmor` | Full armor |
| **Stay In Vehicle** | `.StayInVehicle` | Don't fall off when shot |

### Model Options

- **Change Player Model**: Switch to any ped model
- **Reset Model**: Return to default character
- **Random Model**: Randomize appearance

<Callout type="info">
  Changes to player options are typically temporary and reset on respawn or reconnect, unless configured otherwise.
</Callout>

## Vehicle Options

**Permission**: `vMenu.VehicleOptions`

Modify your current vehicle.

### Quick Actions

| Action | Permission | Description |
|--------|------------|-------------|
| **Repair** | `.Fix` | Instantly repair vehicle |
| **Wash** | `.Wash` | Clean the vehicle |
| **Toggle Engine** | `.Engine` | Start/stop engine |
| **Flip Vehicle** | `.All` | Flip vehicle upright |

### Visual Modifications

**Permission**: `vMenu.VehicleOptions.Mod`

- **Primary Color**: Change paint color
- **Secondary Color**: Change secondary paint
- **Pearlescent**: Add pearlescent finish
- **Wheel Color**: Change wheel color
- **Trim Color**: Interior trim color
- **Dashboard Color**: Dashboard color
- **Liveries**: Apply liveries/wraps
- **Neon Lights**: Enable/customize underglow
- **Tinted Windows**: Window tint level

### Performance Modifications

- **Engine**: Upgrade level 0-4
- **Brakes**: Upgrade level 0-3
- **Transmission**: Upgrade level 0-3
- **Suspension**: Lower or stock
- **Armor**: Protection level 0-5
- **Turbo**: Enable turbo charger

**Permissions for modification**:
- `.Torque`: Modify engine torque multiplier
- `.Power`: Modify engine power multiplier

### Vehicle Components

**Permission**: `vMenu.VehicleOptions.Components`

- **Spoilers**: Add/change spoiler
- **Front Bumper**: Custom front bumper
- **Rear Bumper**: Custom rear bumper
- **Side Skirts**: Add side skirts
- **Exhaust**: Custom exhaust
- **Hood**: Custom hood/bonnet
- **Roof**: Modify roof
- **Grille**: Custom grille

### Door Management

**Permission**: `vMenu.VehicleOptions.Doors`

- Open/close individual doors
- Remove individual doors
- Fix broken doors

### Window Management

**Permission**: `vMenu.VehicleOptions.Windows`

- Roll up/down windows
- Break specific windows
- Repair broken windows

### Special Options

| Option | Permission | Description |
|--------|------------|-------------|
| **Freeze Vehicle** | `.Freeze` | Lock vehicle in place |
| **Invisible Vehicle** | `.Invisible` | Make vehicle invisible (you remain visible) |
| **Engine Always On** | `.EngineAlwaysOn` | Engine never turns off |
| **No Siren Sound** | `.NoSirenSound` | Disable siren audio |
| **No Helmet** | `.NoHelmet` | Disable auto helmet on bikes |
| **Flash Highbeams** | `.FlashHighbeamsOnHonk` | Flash lights when honking |

## Saved Vehicles

**Permission**: `vMenu.SavedVehicles`

Save and spawn custom configured vehicles.

### Features

- **Save Current Vehicle**: Save vehicle with all mods
- **Spawn Saved Vehicle**: Spawn any saved vehicle
- **Delete Saved Vehicle**: Remove from saved list
- **Rename Vehicle**: Change saved vehicle name
- **Replace Vehicle**: Update saved vehicle with current

### Saved Data Includes

- Vehicle model
- All modifications (engine, brakes, etc.)
- Paint colors
- Livery
- Wheels and tires
- Neon lights
- Window tint
- License plate
- Extra components

<Callout type="info">
  Maximum saved vehicles per player is configurable via `vMenuMaxSavedVehicles` convar (default: 50).
</Callout>

## Personal Vehicle

**Permission**: `vMenu.PersonalVehicle`

Designate one vehicle as your personal vehicle with special features.

### Features

- **Set Personal Vehicle**: Make current vehicle personal
- **Recall Vehicle**: Teleport vehicle to you
- **Lock/Unlock**: Remote lock control
- **Return To Storage**: Send vehicle to virtual garage
- **Respawn Vehicle**: Respawn if destroyed

### Exclusive Vehicle

- Only you can drive your personal vehicle (configurable)
- Vehicle persists across resource restarts
- Auto-respawns when destroyed (optional)

## World Options

**Permission**: `vMenu.WorldOptions`

Control server environment.

### Time Control

**Permission**: `vMenu.WorldOptions.Time`

- Set specific time (00:00 - 23:59)
- Freeze time
- Lock time (sync with all players)
- Time sync toggle

### Weather Control

**Permission**: `vMenu.WorldOptions.Weather`

Available weather types:
- Extra Sunny
- Clear
- Clouds
- Overcast
- Rain
- Thunder
- Fog
- Snow
- Blizzard
- Christmas
- Halloween

**Features**:
- Instant weather change
- Weather sync (all players)
- Remove fog/clouds
- Dynamic weather toggle

### World Modifications

**Permission**: `vMenu.WorldOptions.All`

- **Blackout**: Remove all lights
- **Disable Traffic**: Remove all AI vehicles
- **Disable Peds**: Remove all pedestrians
- **Randomize Peds**: Randomize NPC models

<Callout type="warn">
  World changes affect all players on the server. Use responsibly during active gameplay.
</Callout>

## Voice Chat

**Permission**: `vMenu.VoiceChat`

Control voice chat settings.

### Features

- **Enable/Disable Voice**: Toggle voice chat
- **Staff Channel**: Private admin voice channel
- **Proximity Range**: Adjust voice distance
- **Show Speaker Indicators**: Display who's talking

<Callout type="info">
  Requires a voice chat resource like TokoVOIP, Mumble-VOIP, or pma-voice to be installed and configured.
</Callout>

## Misc Settings

**Permission**: `vMenu.MiscSettings`

General preferences and utilities.

### Game Settings

- **Respawn Location**: Choose spawn point
- **Death Notifications**: Toggle death messages
- **Join/Quit Notifications**: Player connection messages
- **Show Coordinates**: Display X, Y, Z on screen
- **Show Location**: Display street name
- **Show Speedometer**: Vehicle speed display
- **Speed Unit**: MPH or KPH

### HUD Settings

- **Hide HUD**: Remove all HUD elements
- **Hide Radar**: Remove minimap
- **Lock Minimap**: Lock minimap rotation
- **Minimap Style**: Normal or expanded

### Player Preferences

- **Save Preferences**: Remember settings
- **Clear Saved Data**: Reset all saved data
- **Keybind Settings**: Change menu keybinds

## Recording Options

**Permission**: `vMenu.Recording`

Rockstar Editor controls.

### Features

- **Start Recording**: Begin recording
- **Stop Recording**: End recording
- **Save Clip**: Save recorded footage
- **Discard Recording**: Delete current recording
- **Start/Stop Editor**: Open Rockstar Editor

### Camera Controls

- Toggle cinematic camera mode
- Adjust field of view
- Camera shake effects

<Callout type="info">
  Recorded clips are saved locally and can be edited in the Rockstar Editor.
</Callout>

## NoClip Mode

**Permission**: `vMenu.NoClip`

Free camera movement for admins.

### Controls

| Key | Action |
|-----|--------|
| **F2** | Toggle NoClip on/off |
| **W/S** | Move forward/backward |
| **A/D** | Move left/right |
| **Space/Ctrl** | Move up/down |
| **Shift** | Increase speed |
| **Mouse** | Look around |

### Settings

- Adjustable speed multiplier
- Configurable keybind
- Invisible while in noclip (optional)

<Callout type="warn">
  **Admin Only**: NoClip is extremely powerful and should only be granted to trusted administrators.
</Callout>

### Features

## Reports System

**Permission**: `vMenu.Report`

Player report system for staff.

### For Players

- Submit reports about other players
- Include reason and description
- Automatic reporter ID included

### For Staff

- View all active reports
- Teleport to reporter/reported player
- Close/resolve reports
- Ban directly from report
