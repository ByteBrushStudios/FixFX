## Overview

The Artifacts API is a high-performance service providing programmatic access to FiveM and RedM server artifacts. It helps server administrators and developers:

- Automate server updates and installations
- Monitor version support status
- Filter artifacts by platform and support level
- Paginate through large artifact datasets
- Access comprehensive artifact metadata

Each artifact includes:
- Version numbers and commit hashes
- Platform-specific downloads (Windows/Linux)
- Support status (recommended, latest, active, deprecated, eol)
- Release dates and file sizes
- Direct download URLs

## Base URL

**Production:** `https://core.fixfx.wiki/api/artifacts`

**Development:** `http://localhost:3001/api/artifacts`

## Authentication

The Artifacts API is public and doesn't require authentication.

## Endpoints

### GET /api/artifacts/fetch

Retrieves paginated list of artifacts with comprehensive filtering options.

#### Query Parameters

| Parameter | Type | Description | Default |
|-----------|------|-------------|---------|
| `platform` | string | Filter by platform (`windows` or `linux`) | `windows` |
| `status` | string | Filter by support status (`recommended`, `latest`, `active`, `deprecated`, `eol`) | All statuses |
| `search` | string | Search by version number (partial match supported) | No search |
| `sortBy` | string | Sort field (`version` or `date`) | `version` |
| `sortOrder` | string | Sort direction (`asc` or `desc`) | `desc` |
| `limit` | number | Results per page (max 100) | `10` |
| `offset` | number | Pagination offset | `0` |
| `includeEol` | boolean | Include End-of-Life artifacts | `true` |

#### Response Format

```json
{
  "data": [
    {
      "version": "24769",
      "hash": "315823736cfbc085104ca0d32779311cd2f1a5a8",
      "platform": "windows",
      "url": "https://runtime.fivem.net/artifacts/fivem/build_server_windows/master/24769/server.zip",
      "size": 52428800,
      "date": "2026-01-25T09:03:21Z",
      "supportStatus": "recommended"
    }
  ],
  "metadata": {
    "total": 150,
    "limit": 10,
    "offset": 0,
    "hasMore": true,
    "platforms": ["windows", "linux"],
    "supportStatuses": ["recommended", "latest", "active", "deprecated"],
    "query": {
      "platform": "windows",
      "status": "",
      "limit": 10,
      "offset": 0,
      "sortBy": "version",
      "sortOrder": "desc",
      "includeEol": true
    }
  }
}
```

### GET /api/artifacts/version/:version

Fetch a specific artifact by version number.

#### Response Format

```json
{
  "data": [
    {
      "version": "24769",
      "hash": "315823736cfbc085104ca0d32779311cd2f1a5a8",
      "platform": "windows",
      "url": "...",
      "size": 52428800,
      "date": "2026-01-25T09:03:21Z",
      "supportStatus": "recommended"
    }
  ],
  "metadata": {
    // ...metadata
  }
}
```

### GET /api/artifacts/check

Check if an artifact version exists and get availability information.

#### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `version` | string | Version number to check |

#### Response Format

```json
{
  "success": true,
  "version": "24769",
  "available": true,
  "platformCount": 2,
  "platforms": [
    {
      "version": "24769",
      "platform": "windows",
      "supportStatus": "recommended"
    },
    {
      "version": "24769",
      "platform": "linux",
      "supportStatus": "recommended"
    }
  ]
}
```

### GET /api/artifacts/changes

Get changelog information comparing two artifact versions.

#### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `base` | string | Base/older version |
| `head` | string | Head/newer version |

#### Response Format

```json
{
  "success": true,
  "base": "24500",
  "head": "24769",
  "comparison": {
    "base_hash": "...",
    "head_hash": "...",
    "base_date": "2026-01-20T...",
    "head_date": "2026-01-25T...",
    "platforms": 2
  },
  "message": "Use GitHub API to fetch detailed commit history"
}
```
