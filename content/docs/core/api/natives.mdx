  functions. Built with Go and Fiber, it fetches from multiple sources (GTA5,

## Overview

The Natives API is a high-performance service for accessing native function documentation across FiveM and RedM. It provides:

- Complete native function databases from multiple sources
- Support for GTA V, RDR3, and CitizenFX natives
- Full-text search across all natives
- Detailed parameter and return type information
- Environment filtering (client, server, shared)
- Cross-reference support between different environments

## Base URL

**Production:** `https://core.fixfx.wiki/api/natives`

**Development:** `http://localhost:3001/api/natives`

## Authentication

The Natives API is public and doesn't require authentication.

## Endpoints

### GET /api/natives

Retrieves native function information with comprehensive filtering and pagination.

#### Query Parameters

| Parameter     | Type    | Description                                          | Default          |
| ------------- | ------- | ---------------------------------------------------- | ---------------- |
| `game`        | string  | Target game (`gta5`, `rdr3`, `cfx`)                  | All games        |
| `search`      | string  | Search query for native names or descriptions        | No search        |
| `ns`          | string  | Filter by namespace (e.g., `PLAYER`, `VEHICLE`)      | All namespaces   |
| `environment` | string  | Filter by environment (`client`, `server`, `shared`) | All environments |
| `cfx`         | boolean | Include CitizenFX-specific natives                   | `false`          |
| `limit`       | number  | Results per page (max 100)                           | `20`             |
| `offset`      | number  | Pagination offset                                    | `0`              |
| `sortBy`      | string  | Sort field (`name`, `namespace`)                     | `name`           |
| `sortOrder`   | string  | Sort direction (`asc` or `desc`)                     | `asc`            |

#### Response Format

```json
{
  "data": [
    {
      "name": "GET_ENTITY_HEALTH",
      "hash": "0xebd235cf",
      "jhash": "0x12345678",
      "ns": "ENTITY",
      "description": "Get the health of an entity",
      "params": [
        {
          "name": "entity",
          "type": "Entity",
          "description": "The entity handle"
        }
      ],
      "results": "int",
      "resultsDescription": "The entity's health",
      "environment": "shared",
      "game": "gta5",
      "isCfx": false,
      "category": "entity",
      "apiset": ""
    }
  ],
  "metadata": {
    "total": 3500,
    "limit": 20,
    "offset": 0,
    "hasMore": true,
    "environmentStats": {
      "client": 1200,
      "server": 800,
      "shared": 1500,
      "total": 3500
    }
  }
}
```

### GET /api/natives/search

Full-text search across all native functions with relevance scoring.

#### Query Parameters

| Parameter | Type   | Description                            | Default   |
| --------- | ------ | -------------------------------------- | --------- |
| `q`       | string | Search query                           | Required  |
| `game`    | string | Filter by game (`gta5`, `rdr3`, `cfx`) | All games |
| `limit`   | number | Results per page                       | `20`      |
| `offset`  | number | Pagination offset                      | `0`       |

#### Response Format

```json
{
  "data": [
    {
      "name": "GET_ENTITY_HEALTH",
      "hash": "0xebd235cf",
      "relevance": 0.95
      // ... full native object
    }
  ],
  "metadata": {
    "query": "health",
    "total": 45,
    "limit": 20,
    "offset": 0
  }
}
```

### GET /api/natives/:hash

Get a specific native by its hash value.

#### Response Format

```json
{
  "data": [
    {
      "name": "GET_ENTITY_HEALTH",
      "hash": "0xebd235cf"
      // ... full native object
    }
  ],
  "metadata": {}
}
```

### GET /api/natives/stats

Get statistics about the natives database.

#### Response Format

```json
{
  "success": true,
  "stats": {
    "total": 3500,
    "byGame": {
      "gta5": 2800,
      "rdr3": 700,
      "cfx": 50
    },
    "byEnvironment": {
      "client": 1200,
      "server": 800,
      "shared": 1500
    },
    "namespaces": 85,
    "lastUpdated": "2026-01-25T09:00:00Z"
  }
}
```

## Usage Examples

### Search for natives by name

```javascript
const response = await fetch(
  "https://core.fixfx.wiki/api/natives/search?q=player",
);
const data = await response.json();
console.log(`Found ${data.metadata.total} natives matching 'player'`);
```

### Get natives for specific namespace

```javascript
const response = await fetch(
  "https://core.fixfx.wiki/api/natives?ns=PLAYER&game=gta5&limit=20",
);
const data = await response.json();
data.data.forEach((native) => {
  console.log(`${native.name} - ${native.description}`);
});
```

### Filter by environment

```javascript
const response = await fetch(
  "https://core.fixfx.wiki/api/natives?environment=client&limit=50",
);
const data = await response.json();
console.log(`Retrieved ${data.data.length} client-side natives`);
```
